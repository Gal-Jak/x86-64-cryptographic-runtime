# x86-64 Crypto Runtime

exploring low-level cryptographic and systems programming on Linux through x86-64 assembly and C.

Focus:
- x86-64 calling conventions
- register and stack behavior
- assembly implementations of selected routines
- C vs assembly comparisons
- cryptographic building blocks at machine level
- constant-time and performance-oriented experiments

Goal is to understand how cryptographic code is actually executed by the CPU and how low-level implementation details affect correctness, security, and performance.
